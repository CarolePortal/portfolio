Hey ! Moi c’est Carole PORTAL

UX UI designer

Chaque jour, nous utilisons de nombreuses interfaces. Et avouons-le, quoi de plus satisfaisant qu'une interface intuitive, esthétique avec une navigation optimale ! Forte de ce constat, le métier d’ux ui designer est venu répondre à ce besoin. Facilitons ensemble la vie des utilisateurs.

Mes domaines de compétences

Découvrez les différentes compétences que je peux vous apporter afin d’améliorer l’expérience de vos utilisateurs.

UX research

Conduite d’entretien, création de persona, UX map afin d’analyser et de synthétiser les usages réels.

UI design

Réalisation d’interfaces : wireframing, création graphique ( création de sites, conception de nouvelles fonctionnalités)

Atelier co-conception

Utilisation de la méthode du design thinking : icebraker, warming up, persona, user journey.

Gestion de projets

Pilotage de projets en mode agile : Lean UX, Scrum etc. Réalisation de roadmap et création de MVP.
